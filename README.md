# 📊 E-Commerce Customer Analytics Platform

![Python](https://img.shields.io/badge/Python-3.10%2B-0078D4) ![MySQL](https://img.shields.io/badge/MySQL-Database-4479A1) ![scikit-learn](https://img.shields.io/badge/scikit--learn-Machine_Learning-F7931E) ![Power BI](https://img.shields.io/badge/Power_BI-Data_Viz-F2C811) ![Pandas](https://img.shields.io/badge/Pandas-2.0%2B-150458) ![NumPy](https://img.shields.io/badge/NumPy-1.24%2B-013243) ![License](https://img.shields.io/badge/License-MIT-success)

An end-to-end Data Analytics project built using SQL, Python, and Power BI on the Brazilian E-Commerce (Olist) Dataset.

This project demonstrates the complete analytics workflow, from raw data cleaning to business intelligence dashboards, customer segmentation, and predictive sales forecasting.

---

## 🚀 Project Objectives

* **Analyze** customer purchasing behavior and trends.
* **Identify** top-performing products, categories, and sellers.
* **Monitor** sales and revenue trajectories over time.
* **Segment** customers using RFM (Recency, Frequency, Monetary) Analysis.
* **Forecast** future revenue using machine learning.
* **Build** interactive, executive-level Power BI dashboards for data-driven decision-making.
